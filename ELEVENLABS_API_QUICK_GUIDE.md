# ElevenLabs API 키 찾기 가이드

## 빠른 해결 방법

### 1. 직접 URL로 접근 (가장 빠름) ⭐

브라우저 주소창에 다음 URL을 입력하세요:

```
https://elevenlabs.io/app/settings/api-keys
```

또는

```
https://elevenlabs.io/app/user/api-keys
```

### 2. Settings 페이지에서 찾기

1. 현재 Settings 페이지에서 상단 탭 확인
2. "Profile" 옆에 **"API Keys"** 또는 **"API"** 탭이 있는지 확인
3. 탭 클릭

### 3. 사이드바에서 찾기

1. 왼쪽 사이드바에서 **"Developers"** 메뉴 찾기
2. "Developers" 클릭
3. API Keys 섹션 확인

### 4. 프로필 메뉴에서 찾기

1. 우측 상단 프로필 아이콘 클릭
2. 드롭다운 메뉴에서 **"API Keys"** 또는 **"Developers"** 찾기

## API 키가 보이지 않는 경우

### 확인 사항

1. **계정 인증 완료**
   - 이메일 인증이 완료되었는지 확인
   - 로그인 상태 확인

2. **무료 플랜 확인**
   - 무료 플랜에서도 API 키 발급 가능
   - "Current Plan: Free"로 표시되어 있어도 가능

3. **브라우저 새로고침**
   - Ctrl+R (Windows) 또는 Cmd+R (Mac)
   - 또는 시크릿 모드에서 시도

4. **다른 브라우저에서 시도**
   - Chrome, Edge, Firefox 등

## 대안: API 키 없이 사용하기

ElevenLabs API 키가 없어도 프로젝트는 작동합니다:

- ✅ **텍스트 채팅**: 정상 작동 (Gemini API만 필요)
- ✅ **음성 인식**: 정상 작동 (Web Speech API 사용)
- ⚠️ **음성 재생**: Mock 모드 (음성 없이 텍스트만 표시)

음성 기능이 필요하지 않다면 ElevenLabs API 키 없이도 사용 가능합니다.
